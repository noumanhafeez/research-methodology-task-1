# Prompt Engineering Demonstration with ChatGPT

## Overview
This repository contains a summary and explanation of a practical demonstration of ChatGPT used in the course **Understanding Prompt Engineering** by Alex Banks.

The goal of this task is to understand how prompt engineering helps users interact effectively with large language models like ChatGPT.

This work was completed as part of a **Research Methodology assignment**.

---

## Objectives

The main objectives of this task are:

- Understand the concept of **Prompt Engineering**
- Learn how prompt quality affects ChatGPT outputs
- Explore how ChatGPT can perform **multiple tasks in a single prompt**
- Demonstrate how prompts can be improved through **iteration**

---

## Key Concepts

### 1. Prompt Engineering
Prompt engineering refers to designing clear and detailed instructions for AI models to obtain better responses.

A basic prompt produces a basic answer, while a detailed prompt produces a more informative response.

---

### 2. Iterative Prompting
ChatGPT responses may not always be perfect on the first try. Users can refine prompts by adding more context or instructions to improve results.

Example:

Basic Prompt: Give me foods to lose fat.

Improved Prompt: Give me foods that help lose fat and gain muscle. Also provide a shopping list and cooking instructions.


---

### 3. Multi-Task Capabilities of ChatGPT
ChatGPT can handle multiple types of tasks in one prompt, including:

- Content generation
- Creative writing
- Coding
- Web design
- Summarization

This shows the flexibility of large language models.

---

## Practical Example from the Lecture

The demonstration shows several stages:

1. Asking ChatGPT for a list of foods for fitness goals.
2. Improving the prompt to include cuisine combinations, shopping lists, and recipes.
3. Moving to creative tasks like writing Shakespeare-style poetry.
4. Asking ChatGPT to generate a webpage design.

This illustrates how prompts can expand from **simple tasks to complex multi-layered outputs**.

---

## Conclusion

Prompt engineering is an important skill when working with AI systems like ChatGPT. By providing detailed instructions and iteratively refining prompts, users can obtain high-quality responses for both practical and creative tasks.

This task demonstrates how effective prompting can significantly enhance the capabilities of AI tools.
